# Kubernetes cluster

### Components
- Cluster TLS issuer
- Hashicorp Vault
- KeyDB (Redis)
- Stolon (PostgreSQL)
- [Flame](https://github.com/pawelmalak/flame)
- Wireguard

    - Remotely to cluster
- Keycloak
- [Trilium](https://github.com/zadam/trilium)
